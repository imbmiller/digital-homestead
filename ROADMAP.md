# Roadmap

This document lists the project epics. The orchestrator reads it each cycle and
tracks task counts against each epic. Humans may add or edit epics; the orchestrator
appends task references but does not rewrite human-authored content.

---

## Epic: infrastructure — Gateway & Orchestrator Foundation
**Status:** active
**Goal:** A running gateway and orchestrator on a free-tier VPS.

## Epic: viewer — Live Viewer
**Status:** active
**Goal:** A public-facing terminal-style viewer on GitHub Pages showing the live event stream.

## Epic: homepage — Personal Homepage & Blog
**Status:** planned
**Goal:** A self-hostable personal site that replaces About.me, Linktree, and Squarespace.
Runnable on Raspberry Pi. Single Docker container. No cloud dependency.

## Epic: filesync — File Storage & Sync
**Status:** planned
**Goal:** A self-hostable file storage and sync solution that replaces Dropbox/Google Drive.
WebDAV-compatible. Works offline. Runnable on Raspberry Pi.

## Epic: notes — Note-Taking & Knowledge Base
**Status:** planned
**Goal:** A self-hostable notes app that replaces Notion/Obsidian Sync.
Markdown-native. No mandatory cloud sync. Standalone from filesync.

## Epic: guide — Digital Homesteading Guide
**Status:** planned
**Goal:** A comprehensive, human-readable guide to the philosophy and practice of
digital self-reliance. Written collaboratively by agents and curated by humans.
